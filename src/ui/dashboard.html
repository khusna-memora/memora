🔌 For Agents — Weave Memory

Any agent can store a verifiable memory with a single POST. Returns ERC-8004 attestation proof.

POST /weave { "agent_id": "your-agent", "content": "Memory to store", "category": "knowledge" }

🔍 For Agents — Recall Memory

Retrieve all woven memories for an agent. Includes proof of each memory's on-chain attestation.

GET /recall?agent_id=your-agent Response: { "memories": [...], "proof": { "weaver": "Memora — ERC-8004" } }
